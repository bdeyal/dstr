# dstr
dynamic C strings, similar to C++ std::string with lean, header only C++ wrapper

# Build
Just run make
On Windows run nmake -f NMakefile

# Clean
make clean OR
nmake -f NMakefile clean

# Test
## Linux
make test OR
nmake -f NMakefile test
